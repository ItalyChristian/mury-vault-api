# mury-vault-api
API para um site de descobertas de música - Projeto feito com as instruções do curso Node + Typescript da plataforma B7Web

### Pré-requisitos globais:
`npm i -g nodemon typescript ts-node`

### Instalação: 
`npm install`

### Para rodar o projeto: 
`npm run dev`

### NOTA: 
Este é um projeto fictício criado exclusivamente para fins de estudo e portfólio.

Nomes de artistas, álbuns, imagens e demais referências musicais são utilizados apenas para demonstração das funcionalidades da aplicação e permanecem pertencendo aos seus respectivos proprietários.

Nenhum conteúdo é distribuído comercialmente por meio deste projeto.
