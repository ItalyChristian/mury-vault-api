# mury-vault-api
API para um site de descobertas de música - Projeto feito com as instruções do curso Node + Typescript da plataforma B7Web

### Pré-requisitos globais:
`npm i -g nodemon typescript ts-node`

### Instalação: 
`npm install`

### Para rodar o projeto: 
`npm run dev`
